## ✨ About The Project

> *"Live life to the fullest. Shatter boredom and embrace your inner kid."* — SPYLT

This project is a **pixel-perfect frontend clone** of the award-winning [SPYLT](https://spylt.com) website — a lactose-free, protein-packed, caffeinated chocolate milk brand that took the web design world by storm when it won **Awwwards Site of the Day (SOTD)**.

Built as a deep-dive learning exercise to master **GSAP ScrollTrigger**, **clip-path animations**, and **scroll-driven UI storytelling** — the exact techniques powering today's most celebrated creative web experiences.

### 🏆 About the Original Brand

| | |
|---|---|
| 🗓️ Founded | 2022 — Lehi, Utah, USA |

| 🍫 Flavours | Chocolate, Strawberry, Cookies & Cream |

## 🛠️ Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| ⚛️ **Framework** | React 19 | Component-based UI |
| ⚡ **Build Tool** | Vite 6 | Fast dev server & bundler |
| 🎬 **Animations** | GSAP 3 + ScrollTrigger | Cinematic scroll animations |
| 🎨 **Styling** | Tailwind CSS v4 | Utility-first styling |
| 📱 **Responsive** | react-responsive | Breakpoint handling |
| 🚀 **Deployment** | Vercel | Edge-optimised hosting |
